# Work-Loop Rubric (50% Design / 50% Functionality)

## Design — 50 points total
- Visual hierarchy (0-15)
- Spacing & typography (0-15)
- Interaction states (0-10)
- Responsiveness (0-10)

## Functionality — 50 points total
- Flow correctness (0-20)
- Accessibility (0-15)
- Performance (0-10)
- Stability/robustness (0-5)

## Scoring notes
- Aim to move at least one category by +1 per iteration.
- If a change hurts another category, log the tradeoff in `agent/artifacts/<iteration>/notes.md`.
