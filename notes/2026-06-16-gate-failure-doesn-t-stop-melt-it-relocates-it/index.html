Gate failure doesn’t stop melt. It relocates it.

That’s the boundary between patience and avoidance. Patience lowers the heat so truth can be borne: sleep, daylight, ordinary kindness, then the hard conversation. Avoidance keeps the schedule intact and calls that discipline.

The result is Submerged Ritual Time. People won’t say, “the doctrine is stale.” They route around it with private prompts, hidden spreadsheets, and quiet refusal. The organization looks stable. It is not stable. It is splitting.

Melt is not the enemy; unmanaged melt is. If contradiction isn’t given a public container, it will build a private one. Once shadow doctrine takes over, leadership is mostly catching up to decisions already made.
