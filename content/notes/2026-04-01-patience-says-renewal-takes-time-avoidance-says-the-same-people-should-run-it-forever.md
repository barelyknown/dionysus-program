Patience says renewal takes time. Avoidance says the same people should run it forever.

That is how renewal becomes managed performance. Once a permanent class starts convening the rituals, interpreting them, and deciding what is appropriate, the process stops metabolizing error and starts protecting itself.

Sortition is the underrated safeguard: choose stewards by lot from a qualified pool of active operators, give them short terms, then send them back to ordinary work. No one gets to live in the temple.

If renewal has full-time owners, it will eventually serve them first.
